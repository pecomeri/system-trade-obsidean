# AI Review Package (FX)

## このパッケージの意図
- Obsidian Vault 全体ではなく、AIレビューに必要な“最小限の要点”だけを切り出すためのパッケージ。
- ノートはコピーであり、元ファイルは改変しない。

## 使い方（ChatGPT / Claude / Gemini）
1) `tree_fx.txt` を読ませて、FX配下の構造を共有する
2) `review_context.md` の TODO を埋めて、依頼したいレビュー内容を明確化する
3) `samples/` を添付して、代表ノートを読ませる

## 注意点
- 秘匿情報、個人情報、APIキー、認証情報（.env、key、token 等）を含めない。
- 大きなファイルやバイナリは含めない（本パッケージでは >5MB を除外）。
- これは“レビュー用のコピー”であり、正本は Vault 側にある。
